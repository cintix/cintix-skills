# cintix-skills

Custom Claude Code skills. Clone into `~/.claude/skills/` to use them.

## Skills

| Skill | Description |
|---|---|
| `zed-project` | Set up a project for Zed editor — auto-detects Java, .NET/C#, PHP, C++, PlatformIO |
| `modular-hybrid-architecture` | Design or refactor apps into a modular monolith |

## Usage

```bash
git clone https://github.com/cintix/cintix-skills.git ~/.claude/skills/
```

Then in Claude Code: `/zed-project` or `/modular-hybrid-architecture`
